This website is a main webpage for the city of Meridian, MS. It has a home page, history page, and a contact page.

The contact page contains a table with names, positions, and districts for city council members. It represents the
SQL file in the folder. The SQL builds a table for 5 different districts in the city and another table with names
and positions for city council members. The council members table references the district table to pull what district
each council member is assigned to.
